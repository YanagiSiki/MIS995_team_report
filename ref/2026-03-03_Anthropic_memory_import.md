# Anthropic 免費提供記憶匯入工具 — 簡介與連結

- 出處：iThome
- 作者：林妍溱
- 發表日期：2026-03-03
- 原始連結：<https://www.ithome.com.tw/news/174126>

簡介：
Anthropic 將 Claude 的「記憶匯入（memory import）」工具全面開放給所有用戶（含免費與付費方案），支援將其他聊天機器人的對話記憶、偏好與上下文搬到 Claude。流程包括從原聊天機器人匯出原始記錄（建議保留原句與細節）、在 Claude 選擇 Import memory 並貼上內容、再點選 Add to memory；匯入後約 24 小時內生效，並可在 Manage edits 編輯已匯入的記憶。Anthropic 並提供範本提示以協助保留完整資訊，方便備份或跨平台搬遷個人化設定。

重點：

- 支援匯入個人偏好、專案、家人/地點等詳細資訊。
- 建議匯出為 Markdown 以保留格式與細節。
- 匯入完成需時間同步（約 24 小時），並提供編輯管理功能。
- 背景：此舉在競爭與監管情境下釐清使用者遷移與資料掌控的便捷性。
